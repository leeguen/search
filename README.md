# 네이버 api를 이용한 검색 프로젝트

## 프로젝트 설명
- 네이버 api를 이용하여 검색하는 방식에 따라 검색결과가 나오는 프로젝트

## 개발환경
- 운영체제 : window 10
- IDE : Intellij
- jdk : jdk 11
- build tool : gradle

## Dependency
- Spring Boot DevTools
- Spring Web
- Lombok
- thymeleaf

## 구현 기능
- 네이버 api 연동
- 검색 기능
